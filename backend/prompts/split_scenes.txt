Each scene should represent a single, clear idea that builds on the last.
Respond with a JSON array of strings, where each string is the description for one scene.

CONCEPT NAME: {concept_name}
CONCEPT DESCRIPTION: {concept_description}

Example:
[
  "Start by introducing the core idea of a neural network as a series of connected nodes.",
  "Then, show how these connections have weights that determine the strength of the signal.",
  "Finally, explain that learning is the process of adjusting these weights to make better predictions."
]

Now, generate the scenes for the provided concept.
